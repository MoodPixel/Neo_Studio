# Roleplay Guide

## 🎭 Overview

The **Roleplay Tab** is designed to structure worldbuilding, character creation, and narrative workflows into clear systems.

Instead of mixing everything into a single prompt or chat, Neo Studio separates creation, interaction, and story management into dedicated areas.

### Text backend:
Connect Backend

### Scene mode:
Select you working on Roleplay, Novel, Stories, Etc

---

## 🧱 Workflow Structure

The Roleplay Tab is divided into the following main areas:

* Forge
* Scene
* Stories
* Studio

Each area serves a specific role in the roleplay pipeline.

---

## 🧩 Sections

---

### 🔨 Forge

**Purpose:**
Create structured content such as:

* Characters
* Worlds
* Universes
* Lore and legends

This is the foundation layer where all roleplay data is built.

**Typical use:**

* Designing characters
* Creating story settings
* Structuring world rules

---

### 🎬 Scene

**Purpose:**
Run live roleplay or narrative sessions.

This is the interaction layer where:

* Characters are used
* Dialogue is generated
* Stories are written in real-time

**Typical use:**

* Roleplay conversations
* Story writing
* Testing characters in context

---

### 📚 Stories

**Purpose:**
Manage and organize written content.

Includes:

* Active workspace
* Story structure (storyline)
* Archive
* Inspector

**Typical use:**

* Saving stories
* Editing narratives
* Reviewing previous content

---

### 🧪 Studio

**Purpose:**
Advanced configuration and system control.

Includes:

* Guide
* Project
* Source
* Assist
* Advanced
* Libraries
* Compile
* Runtime
* Engine
* Inspector

This is the control layer that connects and configures the entire roleplay system.

---

## 🔄 Common Workflows

### Creating a New Character

1. Go to **Forge**
2. Build character details
3. Save to project

---

### Running a Roleplay Scene

1. Go to **Scene**
2. Load characters/world
3. Start interaction

---

### Writing a Story

1. Use **Scene** for generation
2. Save to **Stories**
3. Edit and organize

---

## ⚠️ Tips

* Build strong data in **Forge** before using Scene
* Keep scenes focused to avoid context overload
* Use **Stories** to organize long content
* Use **Studio** only when needed
