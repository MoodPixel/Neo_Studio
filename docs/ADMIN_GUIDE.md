# Admin Guide

## ⚙️ Overview

The **Admin Tab** is used to manage system-level configurations and external backends.

It connects Neo Studio to tools like:

* ComfyUI
* KoboldCPP
* Other local AI services

---

## 🧱 Workflow Structure

The Admin tab includes:

* Backend Launcher
* Node Manager
* System Settings

---

## 🧩 Sections

---

### 🚀 Backend Launcher

**Purpose:**
Launch external tools directly from Neo Studio.

You can:

* Add executable or `.bat` paths
* Start/stop backends

---

### 🧩 Node Manager

**Purpose:**
Manage ComfyUI custom nodes.

**Typical use:**

* Install required nodes
* Update dependencies

---

### ⚙️ System Settings

**Purpose:**
Configure internal behavior.

---

## 🔄 Common Workflows

### Launching ComfyUI

1. Add path in Admin
2. Click launch
3. Connect backend

---

### Installing Nodes

1. Open Node Manager
2. Install required nodes
3. Restart backend

---

## ⚠️ Tips

* Ensure correct paths for executables
* Use Node Manager for easier setup
* Restart backend after changes

---

## 🚧 Current Limitations

* Some backends require manual setup
* Path configuration must be correct

---

## 🚀 Planned Improvements
