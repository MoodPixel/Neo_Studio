# Assistant Guide

## 🤖 Overview

The **Assistant Tab** provides a flexible chat-based system for interacting with AI using memory, context, and project-based workflows.

It acts as a central tool for ideation, writing, and problem-solving.

### Text backend:
Launch Backend > Connect Backend

### Session lane:
If you need a spesific mode

---

## 🧱 Workflow Structure

The Assistant is structured around:

* Chat
* Projects
* Memory
* Context

---

## 🧩 Sections

---

### 💬 Chat

**Purpose:**
Main interaction interface.

You can:

* Ask questions
* Generate content
* Iterate ideas

---

### 📁 Projects

**Purpose:**
Organize conversations into structured workspaces.

**Typical use:**

* Separate different tasks
* Manage long-term workflows

---

### 🧠 Memory

**Purpose:**
Store and reuse important information.

Includes:

* User context
* Project-specific data

---

### 📦 Context

**Purpose:**
Attach additional data to conversations.

Includes:

* Text
* Files
* Image inputs

---

## 🔄 Common Workflows

### Idea Generation

1. Start a chat
2. Ask for ideas
3. Refine outputs

---

### Project-Based Work

1. Create a project
2. Work within that context
3. Save important outputs

---

## ⚠️ Tips

* Keep conversations focused
* Use projects for separation
* Use memory carefully
