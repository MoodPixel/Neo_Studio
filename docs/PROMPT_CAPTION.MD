# Caption & Prompt Guide

## ✍️ Overview

The **Caption & Prompt Tab** is designed to help generate, refine, organize, and reuse text for creative workflows.

It connects image understanding, prompt writing, and assistant-based refinement into one focused workspace.

### Text backend:
Launch Backend > Connect Backend

### Choose your studio:
You need to work on Prompt or Captioning, select your lane here

---

## 🧱 Workflow Structure

The Caption & Prompt Tab is divided into the following main areas:

* Caption
* Prompt
* Library
* Assistant Bridge

Each area supports a different part of the text creation pipeline.

---

## 🧩 Sections

---

### 🖼️ Caption

**Purpose:**
Generate captions or descriptions from images.

You can use this to:

* Describe an image
* Extract visual details
* Convert image content into usable prompt material

**Typical use:**

* Turning reference images into prompts
* Creating captions for generated images
* Understanding visual composition

---

### ✍️ Prompt

**Purpose:**
Create, edit, and refine prompts.

Includes:

* Prompt drafting
* Prompt cleanup
* Prompt improvement
* Positive and negative prompt support

**Typical use:**

* Writing image generation prompts
* Refining prompt structure
* Preparing reusable prompt text

---

### 📚 Library

**Purpose:**
Store and reuse saved prompts or captions.

**Typical use:**

* Reusing successful prompts
* Saving caption outputs
* Organizing prompt collections

---

### 🔁 Assistant Bridge

**Purpose:**
Send prompts or captions into the Assistant for refinement, expansion, or analysis.

**Typical use:**

* Improving weak prompts
* Turning captions into better creative prompts
* Getting assistant feedback before generating

---

## 🔄 Common Workflows

### Image to Prompt

1. Add image in **Caption**
2. Generate caption
3. Send caption to **Prompt**
4. Refine and save

---

### Prompt Refinement

1. Write prompt in **Prompt**
2. Use Assistant Bridge if needed
3. Save final version to **Library**

---

### Reusing Saved Prompts

1. Open **Library**
2. Select saved prompt
3. Edit or send to Image Tab

---

## ⚠️ Tips

* Use captions as a starting point, not always the final prompt
* Keep reusable prompts clean and organized
* Use Assistant Bridge for rewriting or improving complex ideas
* Save successful prompts for future consistency

---
